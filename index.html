1940s

Lenguaje máquina y la era temprana.

El lenguaje máquina es el primer nivel de programación, constituido por instrucciones en código binario directamente entendidas por el hardware...
1950s

Invención del ensamblador simbólico.

Se introdujo el uso de mnemónicos para reemplazar el código binario. El ensamblador traduce estas instrucciones legibles a código máquina, facilitando la programación.
